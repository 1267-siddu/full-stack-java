In client-server architecture, If multiple clients sending the request to the main server then main server will distribute the request to it's parallel servers to reduce the burdun of main server is called distributed application.


> Enterprise App
-----------------
An application which deals with large business complex logic with the help of middleware services is called enterprise application.

Here middleware services means authentication, autherization, malware production, security and etc.

ex:
	Facebook
	Online shopping websites 
	and etc
